# Flutter Micro Lab 003 • Glow FAB

A premium animated Floating Action Button built with Flutter, featuring smooth micro-interactions, a custom glowing ring, floating motion, and SVG-based visuals.

> Small interaction. Big attention to detail.

---

## ✨ Features

- 🌟 Custom glowing ring using `CustomPainter`
- 💬 SVG chat icon
- ⭕ Animated glowing halo
- 🎈 Smooth floating animation
- 👆 Premium press interaction
- 🎨 Fully customizable colors & size
- 📱 Works on Android, iOS, Web & Desktop
- ♻️ Reusable widget

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/TanviKumbhani/flutter_micro_lab.git
```

Install packages

```bash
flutter pub get
```

Run

```bash
flutter run
```
